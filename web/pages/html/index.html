Tags

Tags in Line - Podem ser utilizadas dentro de blocos

ID: Elemento dentro do CSS que tem mais especificidade. Para identificação no CSS utiliza-se "#"
class: Diferente do ID, pode-se ter mais de uma por tag. Identificado no CSS como "."
p: Tag de texto in line. Algumas alterações de texto podem ser feitas dentro da TAG
strong: Utilizada para semântica. O código entende que o que está entre a tag ganha mais importância na leitura. Em leiitores de acessibilidade, o texto é interpretado com mais ênfase.
i: Deixa o texto em itálico Exemplo
em: Mesma lógica do strong, mas dá ênnfase ao texto
a: Utilizada para adicionar links. Usa-se junto com o href=" " e configurção de target para definir onde o link será aberto.
img: Utilizada para inserir imagens.
del: Utilizada para remover algo do código de forma semântica. Exemplo.
ins: Utilizada para inserir algo no código de forma semântica. Exemplo.
s: Utilizada para riscar o texto. Exemplo.
u: Utilizada para sublinhar o texto. Sublinhar.
small: Diminuiu o texto. Exemplo.
sup: Coloca o texto acima. Exemplo.
sub: Coloca o texto abaixo. Exemplo.
q: Utilizada para citações e referências, garalmente acompanhada da tag cite.
span: Utilizada para formatar ou manipular textos inline, sem impactar a estrutura na página.

Tags em blocos - Não podem ser utilizadas dentro de outras tags blocos

blockquote: Mesma lógica da tag q, Utiliza-se o atributo cite para adicionar a referência.
hr: A tag cria uma linha para separar o pagrágrafo
code: Tag utilizada para representar código.
pre: Utilizada para manter a formatação original do conteúdo.
div: É um elemento de bloco que quebra o parágrafo.
nav: Utilizada para criar uma área de nagevação com links.
ul: Utilizada para criar lista não ordenada. Para cada tópico utiliza-se li.

Separação de conteudo - Tags utilizadas para determinar conteudos.

address: Utilizado para para colocar endereços, aqueles que ficam no final da página
article: Utilizado para artigos em site de blogs e etc, posso utilizar também para produtos
aside: Barra lateral
footer: Rodapé do site, o footer pode ser utilizados em outros blocos
header: Cabeçalho do site

Listas

ul Utilizado para listas não ordenadas.
  • Item 1
  • Item 2
ol Utilizado para lista ordenada. Diferente de ul, possui um identificador de ordem (1, 2, 3...). Pode ser alterado com atributos como type
  1. item 1
  2. item 2
dl Utilizado para listas com descrição. Usa-se dt para o termo e dd para a descrição.
