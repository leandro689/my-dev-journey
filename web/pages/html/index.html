Tags

Tags in Line - Podem ser utilizadas dentro de blocos

ID: Elemento dentro do CSS que tem mais especifidade, para identificação no CSS utiliza-se "#"
class: Diferente do ID, pode-se ter mais de um por tag, identificado no CSS como "."
p: Tag de texto in line, algumas alterações de texto podem ser dentro da TAG
strong: Utilizada para semantica, o cod vai entender que o que está entre a tag ganha força na leitura. Para ficar mais claro, caso haja um leitor de cod para acessibilidade ele vai interpretar o texto com maior força.
i: deixa o texto em italico Exemplo
em: mesma lógica do Strong, mas dá enfase ao texto
a: utilizada para adicionar link dentro da tag, utiliza-se em conjunto com o href=" " para adicionar o link e configurar os targets para abrir o link na mesma guia ou em guias diferentes.
img: Utiliza-se para colocar imagens.
del: Utiliza-se para remover algo do cod de forma semantica. Exemplo.
ins: Utiliza-se para inserir algo no cod de forma semantica. Exemplo.
s: Utiliza-se para riscar o texto. Exemplo.
u: Utiliza-se para sublinhar o texto. Sublinhar.
small: Diminuiu o texto. Exemplo.
sup: Coloca o texto em cima. Exemplo.
sub: Coloca o texto em baixo. Exemplo.
q: Utiliza-se para citação, referencias e etc, para evitar stryke, tag acompanhada ta tag cite para citar de onde vem a info.
span: Utiliza-se para formatar ou manipular textos in line, sem impactar na estrutura da pag.

Tags em blocos - Não podem ser utilizadas dentro de outras tags blocos

blockquote: Mesma lógica do q, utiliza-se o atributo cite para cadastrar a ref dentro da tag.
hr: a tag cria uma linha para separar o pagragrafo, como feito à cima.
code: tag utilizada para informar um código entre tag.
pre: utilizada para utilizar a pré formatação do código, ou o que estiver entre ela.
div: é um bloco, quebra o paragrafo.
nav: Utiliza-se para criar uma area de nagevação, com links e etc.
ui: utilizada para criar lista ordenada, para cada tópico utiliza-se li.

Separação de conteudo - Tags utilizadas para determinar conteudos.

address: Utilizado para para colocar endereços, aqueles que ficam no final da pagina
article: Utilizado para artigos em site de blogs e etc, posso utilizar também para produtos
aside: Barra lateral
footer: Rodapé do site, o footer pode ser utilizados em outros blocos
header: Cabeçalho do site

Listas

ul Utilizado para listas não ordenadas.
  • Item 1
  • Item 2
ol Utilizado para lista ordenada, diferente de ul, ele traz um identificador de ordem. Emxemplo 1, 2, 3 e por ai vai. Pode mudar o identificador com o atributo title=""
  1. item 1
  2. item 2
dl Utilizado para listras onde os item precisam de descrição. utilizando o dl para abrir a lista e utilizamos em seguida o dd para a tag de descrição.
