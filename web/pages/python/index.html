Palavras-Chave / Sintaxe

  • if - estrutura de controle condicional.
  • elif - complemento de if.
  • else - complemento final do if.
  • try - Utiliza para tentar executar algo dentro do código. Caso não consiga, ele vai jogar no Except.
  • except - Utilizado para tratamento de erro dentro do try.
  • pass - Utiliza-se para ignorar condições, caso não seja estruturado nada entre os blocos try ou if, pode utilizar o pass para ignorar o erro que seria devolvido.
  • while - Laço dentro do Python, enquanto a condição a ele estipulada for verdadeira, o código vai continuar se repetindo. Pode sair com o Break ou com a alteração da condição a ele estipulada.
  • break - Pode-se utilizar para quebrar algum algorimto ou algum laço.
  • continue - Utiliza-se para pular uma parte do algoritmo ou algum bloco.
  • and - Mesma lógica da porta and.
  • not - Iverte a condição ou situação de verdadeiro ou falso do objeto.
  • or - Mesma lógica da porta or
  • for - Utiliza-se em objetos iteraiveis, fazendo o código passar em item por item do objeto.
  • del - Utilizada para remover itens de uma estrutura ou deletar uma variavel.

Comentários

  • # - Utiliza-se para comentar a linha dentro do código do Python.
  • '''-''' e """-""" - Aspas simples e aspas dessa forma podem ser utilizadas como caixa de comentário (Docstring), funcionam em mais de uma linha de uma vez.
  • Sintaxes de str - \: Utilizado para escapar um valor dentro de uma str, exemplo print("\"teste\"");

Funções

  • input() - Utiliza-se para cadastrar dados em varíaveis ou objetos no código.
  • id() - Utilizado para mostrar o id de um objeto na memoria dentro do Python.
  • len() - Conta a quantidade caracteres dentro de um objeto.

O py, sempre vai executar o código de cima pra baixo, da direita pra esquerda, os operadores nos ajudam a a decidir o que vai ser executado primeiro dentro da linha do código. Por exemplo:
print(arg1+arg2*(arg1+arg2)) - aqui o parenteses será executado primeiro, depois a multiplicação e depois a soma na linha código.

Índice

Índices, utilizamos para pegar um dado especifico de uma informação, sendo uma STR. Com o índice conseguimos, fatiar strs e exibir só o necessário.
Exemplo: 


                            nome = input('Digite seu nome: ')
                            data_n = input('Digite sua data de nascimento: ')

                            concat = nome + data_n

                            print(
                                f'Seu nome é "{concat[:len(nome)]}"
                                e nasceu em "{concat[len(nome):]}"'
                            )

Observação: Aqui utilizamos inputs para informações, a variável concat pra juntar todas as infos fornecidas e depois utilizamos interpolação pelo f'', para fatiarmos a str. percebe-se que o fatiamento é feito dentro da função "[]". Desta forma: [inicio da str: até onde vai] Espaços vazios definem como 0. Ou seja no trecho do código, "[:len(nome)] ele conta quantas letras tem o input nome inserido e o converte em número, assim vai contar do 0 até a quantidade de caracteres existentes em nome."

Contadores Aritméticos

Contadores aritimeticos são utlizados para acumular ou contar valores.

Os operadores sempre vão fazer a função deles.

Exemplo: variável = 10, variável += 4, teremos o valor 14. Isso ocorreu por conta do acumulo aritimetico (10+4). Essas funções são muito utilizadas em laço pra agregar valor a uma variável ou mais.
  • = - Utilizado para atribuir valor a uma variavel
  • += - Soma valores, ou concatena (em casos de string).
  • *= - Multiplica valores.
  • /= - Divide o valor e converte o item para float.
  • //= - Divide o valor, e traz o valor inteiro, exemplo: 10 // 4 = 2
  • **= - Exponencia o valor.
  • %= - Traz a sobra de uma divisão.

Classes - excepet

Podemos utilizar classes para tratar erros dentro do try/except
ValueError: Utiliza-se para um valor incorreto, conseguimos utilizar em conversões.
IndexError: Utiliza-se para erros em índices, quando por exemplo, tentamos excluir um item de uma lista e utilizamos o ídice incorreto para isso.
Exception: Utiliza-se para quando não sabemos qual erro pode dar dentro do try.

Imprecisão ponto flutuante

Em algumas linguágens de programação, é utilizado um sistema chamado "Double-precision floating-point format IEEE 754."
Em somas de números flutuantes, ao efetuarmos o cálculo por exemplo de 0.1 + 0.7, o resultado seria de 0.7999999.
Existem algumas formas de contornar essa situação, uma delas é utilizando o f'{variavel:.f}'.
A função round() que arredonda o número decimal.
Import decimal do Python, esse é utilizado somente para números flutuantes que precisamos de dados extremamente precisos. decimal.Decimal('0.1')

                                        numero1 = int(numero1) - int(numero2)
                                    elif operacao == "*":
                                        numero1 = int(numero1) * int(numero2)
                                    elif operacao == "/":
                                        numero1 = int(numero1) / int(numero2)
                                    ops += 1
                                    print(f"Resultado da {ops}ª operação: {numero1}")
                                except:
                                    print(f'{numero2} não é um número!')
                            else:
                                print("Operação inválida!")

                        print("Você saiu da calculadora.")

Observação: Comparado com o algoritmo codado por mim, a versão do GPT unifica as operações dentro do if, não sendo necessário um if pra cada operação.
