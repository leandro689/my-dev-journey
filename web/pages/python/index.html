print A função print, utilizamos para imprimir algo na tela, seja uma variavel, uma string, um numero, algum objeto do py

  • Sep - dentro do print, serve para definir um caracter de separação - Ex: print(arg1,arg2,arg3) sem o sep, teremos o seguinte resultado: arg1 arg2 arg3
    Caso tenha o o sep no print: print(arg1,arg2,arg3, sep='-'), teremos o seuinte resultado: arg1-arg2-arg3
  • end='' - dentro do print ele define qual será o final do print, todo print ele ja tem como final um "enter", se definirmos com o end ele não dará o enter e colocará o caracter selecionado
    Exemplo: sem end
    print(arg1)
    print(arg2)
    Resultado:
    arg1
    arg2

    Exemplo: com end
    print(arg1,end='-') print(arg2)
    Resultado:
    arg1-arg2
  • \ utilizado para escapar algo dentro de um texto, por exemplo se queremos colocar algo entre "" dentro deuma string, quando colocamos entr \\, o py vai entender que faz parte do texto e não vai considerar a "" como final da primeira string

                        frase = input('Digite uma frase: ')

                        let_mais_mencionada = ''
                        qte_letra_mais_mencionada = 0
