Estudos CSS

Anotações

width: Define a largura do elemento.
background-image: Define uma imagem de fundo para o elemento.

Box Model

content: Conteúdo interno do elemento.
padding: Define o espaçamento interno entre o conteúdo e a borda.
border: Define a borda ao redor do elemento.
margin: margem da box, ela não preenche, ela afasta do conteúdo
box-sizing: Define como o tamanho total do elemento é calculado.
Com border-box, o width e o heigh incluem padding e border, mantendo o tamanho total fixo.
